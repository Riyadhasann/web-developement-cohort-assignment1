# Setup Verification Assignment

Welcome! This is your setup verification assignment for the Web Development Bootcamp.

## Instructions

1. Clone this repository to your computer
2. Open the folder in VS Code
3. Edit `index.html` and replace:
   - `[YOUR FULL NAME HERE]` with your actual name
   - `[YOUR GITHUB USERNAME]` with your GitHub username
4. Save the file
5. Commit your changes with a meaningful message
6. Push to GitHub
7. Deploy to Vercel
8. Submit your GitHub repo URL and Vercel deployment URL in Google Classroom

Full instructions are available in the Google Classroom assignment.
